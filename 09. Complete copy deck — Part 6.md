## Page 113 — Complete mailbox administration on Windows or the web  
## Complete advanced mailbox administration elsewhere  
Outlook on iPadOS supports common email and follow-up tasks. Some mailbox-management tasks are easier or only available on Windows or Outlook on the web.  

| Task | Recommended platform |
| ------------------------------------ | -------------------------------------------- |
| Create or edit inbox rules | New Outlook or Outlook on the web |
| Reorder or troubleshoot rules | Windows or web |
| Use Sweep | Outlook on the web or New Outlook |
| Create or manage Quick Steps | Windows or web where supported |
| Create or manage Search Folders | Classic Outlook or supported Windows version |
| Restructure several folders | Windows or web |
| Complete large-scale cleanup | Windows or web |
| Manage advanced categories | Windows or web |
| Configure shared-mailbox rules | Windows or web |
| Configure shared automatic replies | Windows or web |
| Manage mailbox permissions | Approved access or support process |
| Configure Shared Sent Items behavior | Administrative support |
  
****Move the task safely****  
Before switching platforms:  
1. Confirm that the iPad has a network connection.  
2. Allow Outlook to synchronize.  
3. Save unfinished drafts.  
4. Confirm the account and mailbox.  
5. Note the exact task you need to complete.  
6. Open the appropriate Windows or web guide.  
7. Complete the advanced procedure.  
8. Return to Outlook on the iPad.  
9. Refresh the mailbox.  
10. Confirm that the change appears.  
## Example — Create a rule  
You receive a predictable automated report every Monday.  
On iPad:  
1. Review the message.  
2. Confirm that no immediate action is required.  
3. Note the sender and subject.  
On Outlook on the web or New Outlook:  
1. Create the rule.  
2. Test it with a low-risk message.  
3. Confirm the destination folder.  
Back on iPad:  
1. Refresh Outlook.  
2. Open the folder.  
3. Confirm that future messages appear as expected.  
## What to remember  
Mobile Outlook may display the result of a mailbox setting even when the setting must be created elsewhere.  
## Avoid this  
* Do not create duplicate organization systems across platforms.  
* Do not assume the iPad can manage every mailbox setting.  
* Do not make mailbox-wide changes without understanding the impact.  
* Do not switch platforms before confirming that drafts and changes synchronized.

## Page 116 — Outlook will not open, load or refresh  
## Outlook does not open  
1. Confirm that the iPad is unlocked.  
2. Confirm that the Outlook app is present.  
3. Tap Outlook once.  
4. Wait for the app to load.  
5. Confirm network connectivity.  
If Outlook does not respond:  
1. Open the app switcher.  
2. Close Outlook.  
3. Reopen it.  
4. Restart the iPad when appropriate.  
## Outlook is missing  
1. Use Spotlight search.  
2. Review the App Library.  
3. Confirm that the device has a connection.  
4. Restart the iPad.  
5. Contact device or Jamf support.  
Do not reinstall Outlook using a personal Apple ID.  
## Mail does not refresh  
1. Confirm Wi-Fi or cellular connectivity.  
2. Open the correct account and Inbox.  
3. Pull down on the message list when supported.  
4. Wait for synchronization.  
5. Close and reopen Outlook.  
6. Compare the mailbox on Windows or the web.  
## Outlook freezes or closes  
1. Save or confirm any draft when possible.  
2. Close Outlook from the app switcher.  
3. Reopen it.  
4. Restart the iPad if the issue continues.  
5. Confirm whether other managed applications work.  
6. Contact support if Outlook continues to close.  
## Confirm the managed update status  
Use the approved company process to confirm that Outlook is current.  
Do not:  
* Delete the application  
* Reinstall it independently  
* Remove the managed account  
* Remove Jamf or management profiles  
## Get help when  
* Outlook remains unavailable after restart  
* The app repeatedly closes  
* Several managed apps fail  
* Outlook is missing from the device  
* The update is blocked or incomplete  
* Mail does not synchronize across platforms  
## Support information to provide  
* iPad model  
* iPadOS version  
* Outlook version  
* Date and time  
* Network type  
* Account  
* Exact behavior  
* Screenshot with sensitive content removed  
* Steps already attempted  
## What to remember  
Refresh first, reopen second and restart only when needed.

## Page 117 — Wrong account or repeated authentication  
## The wrong account or mailbox appears  
1. Open the Outlook account menu.  
2. Confirm the company work account.  
3. Select the intended primary or shared mailbox.  
4. Open the correct folder.  
5. Confirm the sending account before composing.  
## Outlook repeatedly asks you to sign in  
Check:  
* Network connection  
* Company account  
* Device date and time  
* Approved authentication application  
* Whether another Microsoft 365 app also requests sign-in  
* Whether the iPad recently restarted or updated  
Then:  
1. Complete the approved authentication prompt.  
2. Return to Outlook.  
3. Lock and unlock the iPad.  
4. Close and reopen Outlook.  
5. Restart the iPad when appropriate.  
## Authentication fails  
Use [insert authentication support path] when:  
* The account is locked  
* Verification fails repeatedly  
* The prompt shows an unexpected organization  
* Outlook returns to sign-in after successful authentication  
* Verification codes do not work  
* Other managed apps also fail  
## Draft recovery  
After signing back in:  
1. Open **Drafts**.  
2. Review unfinished messages.  
3. Confirm attachments and recipients.  
4. Continue or discard the drafts.  
## What to remember  
Confirm the application, work account, mailbox and folder separately.  
## Avoid this  
* Do not approve a sign-in you did not initiate.  
* Do not share verification codes.  
* Do not remove the managed account.  
* Do not add a personal account to solve a work-account issue.  
* Do not continue working in the wrong mailbox.

## Page 118 — Notification and badge issues  
## Outlook notifications do not appear  
## Check Outlook  
1. Open **Outlook Settings**.  
2. Tap **Notifications**.  
3. Confirm the company account.  
4. Confirm that the intended mail or calendar notifications are enabled.  
## Check iPadOS  
1. Open **Settings**.  
2. Tap **Notifications**.  
3. Tap **Outlook**.  
4. Confirm **Allow Notifications** is on when permitted.  
5. Review:  
    * Lock Screen  
    * Notification Center  
    * Banners  
    * Sounds  
    * Badges  
## Check Focus  
1. Open **Settings > Focus**.  
2. Review the active Focus mode.  
3. Confirm whether Outlook is allowed.  
4. Change the setting only when permitted.  
## The badge count looks incorrect  
Badge counts may reflect:  
* Unread messages  
* Focused Inbox only  
* More than one account  
* Delayed synchronization  
* Organization settings  
Try:  
1. Open Outlook.  
2. Confirm the selected account.  
3. Refresh the Inbox.  
4. Review Focused and Other.  
5. Close and reopen Outlook.  
## Notifications show too much information  
Review:  
* Lock Screen preview settings  
* Outlook notification content  
* iPadOS notification previews  
* Organization-managed restrictions  
## Get help when  
* Managed notification settings cannot be changed  
* Notifications never appear  
* Alerts appear for the wrong account  
* Badge counts remain incorrect after synchronization  
* Sensitive previews appear unexpectedly  
## What to remember  
Outlook, iPadOS and Focus settings can each affect notifications.  
## Avoid this  
* Do not depend on notifications as the only follow-up system.  
* Do not display sensitive previews when restricted.  
* Do not disable required alerts.  
* Do not assume a missing alert means the message did not arrive.

## Page 119 — Keyboard, trackpad and swipe issues  
## The Magic Keyboard does not respond  
1. Confirm that the iPad is seated correctly.  
2. Remove and reconnect it carefully.  
3. Test typing in another approved application.  
4. Restart the iPad.  
5. Review **Settings > General > Keyboard**.  
6. Contact device support if it remains unavailable.  
## The trackpad pointer is missing or difficult to use  
1. Confirm that the Magic Keyboard is connected.  
2. Move one finger across the trackpad.  
3. Open **Settings > General > Trackpad** or **Trackpad & Mouse**.  
4. Review tracking speed.  
5. Review Tap to Click and Secondary Click.  
6. Test the pointer in another approved application.  
## A shortcut does not work  
1. Confirm that the cursor or focus is in the intended Outlook area.  
2. Press and hold **Command** to review available shortcuts.  
3. Test a standard editing shortcut.  
4. Confirm the shortcut was validated for Outlook on iPad.  
5. Close and reopen Outlook.  
## A swipe action does the wrong thing  
1. Stop swiping through messages.  
2. Use Undo when immediately available.  
3. Check Archive, Deleted Items or Snoozed.  
4. Open **Outlook Settings > Swipe Options**.  
5. Review the left and right actions.  
6. Test them using low-risk messages.  
## Determine whether the issue is Outlook-specific  
Test:  
* Typing in Word or Notes  
* Trackpad movement on the Home Screen  
* Copy and paste in another approved app  
* App switching  
If the problem occurs across applications, use device support.  
## What to remember  
Test the hardware in another application before treating the problem as an Outlook issue.  
## Avoid this  
* Do not force the keyboard connector.  
* Do not assume Windows shortcuts apply.  
* Do not change several input settings at once.  
* Do not continue using an unknown swipe action.

## Page 120 — Attachment and file issues  
## An attachment will not open  
Check:  
1. Internet connection  
2. File type  
3. File size  
4. Sender  
5. Whether the file is protected  
6. Whether the approved application is installed  
Then:  
1. Try the Outlook preview.  
2. Open the file in the approved Microsoft application.  
3. Compare the file on Windows or the web.  
4. Contact support if the approved workflow fails.  
## A file cannot be saved  
The destination may be:  
* Unavailable offline  
* Unapproved  
* Restricted by Jamf  
* Read only  
* Missing required permission  
* Full or unavailable  
Use an approved OneDrive, SharePoint or managed Files location.  
## A sharing destination is blocked  
A blocked destination may indicate managed-data protection.  
Do not:  
* Copy the file into a personal app  
* Use personal cloud storage  
* Use AirDrop without approval  
* Change management settings  
* Rename the file to bypass restrictions  
## An offline file is unavailable  
1. Confirm the network connection.  
2. Open OneDrive or the approved source.  
3. Confirm that the file still exists.  
4. Review whether it was previously made available offline.  
5. Download or open it again when permitted.  
## The wrong file version opens  
1. Review the file name.  
2. Review the modified date.  
3. Check the source location.  
4. Search for the shared cloud version.  
5. Remove unnecessary duplicate copies when permitted.  
## Get help when  
* Approved files are consistently blocked  
* Required applications are missing  
* OneDrive or SharePoint locations do not appear  
* A managed restriction prevents required work  
* File permissions appear incorrect  
* The issue occurs on multiple platforms  
## What to remember  
A blocked destination may be an intentional data-protection control.  
## Avoid this  
* Do not bypass managed restrictions.  
* Do not use personal storage.  
* Do not repeatedly download duplicate copies.  
* Do not open unexpected attachments.

## Page 121 — Shared mailbox and Copilot issues  
## The shared mailbox is missing  
1. Confirm the primary company account.  
2. Confirm that access was approved.  
3. Refresh Outlook.  
4. Close and reopen the app.  
5. Add the shared mailbox again when appropriate.  
6. Wait if permission was assigned recently.  
7. Contact shared-mailbox support.  
## The shared sending address is unavailable  
Check:  
* Shared mailbox is selected  
* The message was created from the shared account  
* Send As or Send on Behalf permission  
* Outlook synchronization  
* Whether the address appears on Windows or the web  
## Shared Search is incomplete  
1. Select the shared account.  
2. Clear filters.  
3. Search by exact sender or subject.  
4. Refresh Outlook.  
5. Compare Search in Outlook on the web.  
## Copilot is missing  
Check:  
* Company work account  
* Copilot license  
* Supported primary mailbox  
* Internet connection  
* Outlook version  
* Another supported message  
* Feature availability on Windows or the web  
## A thread cannot be summarized  
Check:  
* Primary mailbox  
* Conversation length  
* Connection  
* Supported protection  
* Another eligible conversation  
## An attachment cannot be summarized  
Check:  
* Supported file type  
* File opens normally  
* Primary mailbox  
* Feature availability on the iPad  
* Windows or web alternative  
## Copilot output is incorrect  
1. Stop before sending or acting.  
2. Open the source.  
3. Review citations.  
4. Remove unsupported statements.  
5. Correct names, dates, owners and deadlines.  
6. Improve the prompt when regenerating.  
7. Complete the final review.  
## What to remember  
Missing mobile controls may reflect licensing, permissions, mailbox type, app version or rollout.  
## Avoid this  
* Do not assume Copilot uses shared-mailbox content.  
* Do not send incorrect output.  
* Do not use unsupported workarounds.  
* Do not remove the managed account to restore a feature.
