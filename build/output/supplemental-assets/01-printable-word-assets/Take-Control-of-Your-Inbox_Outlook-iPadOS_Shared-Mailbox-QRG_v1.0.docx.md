## Shared mailboxes in Outlook on iPadOS  
## Before you start  
You need approved access.  
Permissions may include:  
* Read and manage  
* Send As  
* Send on Behalf  
Adding the mailbox displays it. It does not grant permission.  
  
## Add the shared mailbox  
1. Open Outlook.  
2. Tap the account or profile control.  
3. Tap **Add Account**.  
4. Tap **Add a Shared Mailbox**.  
5. Enter the approved mailbox name or address.  
6. Select the correct result.  
7. Wait for Outlook to add it.  
8. Tap the shared mailbox.  
9. Open **Inbox**.  
  
## Switch between mailboxes  
1. Open the account menu.  
2. Tap the primary or shared mailbox.  
3. Confirm the active account.  
4. Open the required folder.  
Before acting, confirm:  
* Mailbox  
* Folder  
* Message  
* Sending account  
  
## Review shared folders  
Check:  
* Shared Inbox  
* Shared Drafts  
* Shared Sent  
* Team folders  
* Deleted Items  
Before replying, review Shared Sent to see whether someone already responded.  
  
## Send from the shared account  
1. Open the shared mailbox.  
2. Tap **Compose**.  
3. Confirm the shared sending account.  
4. Add recipients.  
5. Enter a clear subject.  
6. Write the message.  
7. Add the approved shared signature.  
8. Review attachments and links.  
9. Confirm the shared account again.  
10. Tap **Send**.

## Page 2 copy  
## Coordinate shared work  
## Before replying  
Check:  
* Has someone already responded?  
* Is another employee handling it?  
* Is a team status applied?  
* Does Shared Sent contain a recent response?  
* Is the shared account selected?  
## Suggested statuses  

| Status        | Meaning                      |
| ------------- | ---------------------------- |
| New           | Not reviewed                 |
| In progress   | Someone owns the work        |
| Waiting       | Waiting for information      |
| Ready to send | Final review needed          |
| Complete      | Response sent or work closed |
| Escalated     | Additional help is required  |
  
Use the team’s established process when one already exists.  
  
## Search the shared mailbox  
1. Select the shared mailbox.  
2. Tap **Search**.  
3. Enter the sender, subject, phrase or file name.  
4. Apply available filters.  
5. Open the correct result.  
6. Confirm the result account.  
7. Confirm the shared sending account before replying.  
  
## Use flags and categories carefully  
A shared flag or category may help the team see status.  
It may not:  
* Assign ownership  
* Create a personal To Do task  
* Notify another employee  
* Add a deadline  
Use the team process.

## Common problems  

| Problem                               | Check first                        |
| ------------------------------------- | ---------------------------------- |
| Mailbox is missing                    | Access, account and refresh        |
| Shared sending account is unavailable | Permission and selected account    |
| Search misses messages                | Shared account and filters         |
| Duplicate replies occur               | Shared Sent and team status        |
| Categories are unavailable            | Mobile limitation or rollout       |
| Rules are unavailable                 | Complete on Windows or web         |
| Sent Items are unexpected             | Shared and personal Sent locations |
  
**Get help when**  
* Approved access does not appear  
* Send As or Send on Behalf fails  
* Shared Sent behavior is incorrect  
* Search repeatedly misses shared email  
* A team-wide setting must change

## What to remember  
Confirm ownership, the selected account and the From address before sending.  
## Avoid this  
* Do not share credentials.  
* Do not sign in directly as the shared mailbox.  
* Do not assume unread means unassigned.  
* Do not change team-wide settings without approval.  
* Do not assume Outlook Copilot uses shared-mailbox content.
