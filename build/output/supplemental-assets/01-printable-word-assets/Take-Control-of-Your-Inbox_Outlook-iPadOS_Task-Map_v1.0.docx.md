## I need to respond or send  

| I need to… | Use |
| ----------------------------------------- | --------------------------------- |
| Answer only the sender | Reply |
| Answer everyone who needs the information | Reply All |
| Share the message with someone new | Forward |
| Start a new topic | Compose |
| Send later | Schedule Send, where available |
| Save unfinished work | Drafts |
| Improve a draft | Formatting or Coaching by Copilot |
| Complete advanced sending options | Windows or Outlook on the web |
  
**Before sending, review:** Account, From address, recipients, subject, action, deadline, files, links, permissions and tone.  
  
## I need to attach or share a file  

| I need to…                        | Use                                 |
| --------------------------------- | ----------------------------------- |
| Attach a recent company file      | Recent files                        |
| Browse company storage            | Files app                           |
| Collaborate on one current file   | OneDrive or SharePoint link         |
| Send a fixed version              | Attach a copy                       |
| Review a received file            | Preview or approved Microsoft app   |
| Save a file                       | OneDrive or approved Files location |
| Send through another approved app | iPad share sheet                    |
  
Use approved company storage. Do not use personal iCloud, personal apps or AirDrop unless specifically approved.  
  
## I need to make follow-up visible  

| I need to…                          | Use                       |
| ----------------------------------- | ------------------------- |
| Track unfinished work               | Flag                      |
| Bring the message back later        | Snooze                    |
| Identify status or work type        | Category, where supported |
| Complete a common action quickly    | Swipe action              |
| Track several steps or a deadline   | Microsoft To Do           |
| Focus on selected work today        | My Day                    |
| Reserve focused work time           | Calendar event            |
| Coordinate a discussion or decision | Meeting                   |
  
**Visible beats remembered.**  
  
## I need to work in a shared mailbox  
1. Open the Outlook account menu.  
2. Select the shared mailbox.  
3. Review Shared Inbox, team status and Shared Sent.  
4. Confirm ownership.  
5. Confirm the shared sending account.  
6. Send or reply.  
7. Update the team status.  
**Confirm ownership, the selected account and the From address before sending.**  
  
## I need help from Copilot  

| I need to… | Use |
| ------------------------------------------- | --------------------------------------------- |
| Catch up on a long conversation | Summarize |
| Review a supported attachment | Summarize a file, where available |
| Create a first draft | Draft with Copilot |
| Improve tone or clarity | Coaching by Copilot |
| Use broader Copilot Chat | Approved Windows, web or Copilot app workflow |
| Schedule or manage advanced mailbox actions | Windows or Outlook on the web |
  
**Review Copilot output for**  
* Accuracy  
* Sources  
* Selected account  
* From address  
* Recipients  
* Owners  
* Dates and deadlines  
* Tone  
* Completeness  
* Attachments and links  
* Permissions  
* Sensitive details  
* Final wording  
**Copilot starts the draft. You finish the message.**
