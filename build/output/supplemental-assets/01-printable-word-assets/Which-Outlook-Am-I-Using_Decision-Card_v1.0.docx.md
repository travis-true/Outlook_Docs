## Copy deck  
## Which Outlook am I using?  
## Are you using the Outlook app on an iPad?  
## Yes  
## Outlook on iPadOS  
Look for:  
* iPad status bar  
* Outlook mobile app  
* Touch-first controls  
* Apple Magic Keyboard support  
* No browser address bar  
* No Windows ribbon  
**Use:** Outlook on iPadOS guide  
## No  
Continue below.  
  
## Is Outlook open inside Edge or Chrome?  
## Yes  
## Outlook on the web  
Look for:  
* Browser tab  
* Address bar  
* Browser profile icon  
* Outlook inside a webpage  
**Use:** Outlook on the web guide  
  
## Is Outlook a separate Windows application with Settings in the upper-right area?  
## Yes  
## New Outlook for Windows  
Look for:  
* Modern interface  
* Separate Windows app  
* No browser address bar  
* No File or Send/Receive ribbon tabs  
**Use:** New Outlook for Windows guide  
  
## Do you see File and Send/Receive tabs?  
## Yes  
## Classic Outlook for Windows  
Look for:  
* Traditional ribbon  
* File tab  
* Send/Receive tab  
* Desktop Outlook windows  
**Use:** Classic Outlook for Windows guide
