## Search and gestures in Outlook on iPadOS  
## Search faster  
1. Confirm the account.  
2. Tap **Search**.  
3. Enter the most distinctive detail.  
4. Apply available filters.  
5. Confirm the result account.  
6. Clear the search when finished.  
## Useful searches  

| I need to find…            | Search or filter            |
| -------------------------- | --------------------------- |
| Email from a person        | Sender name or address      |
| Email sent to a person     | Recipient name              |
| Subject contains a phrase  | Subject words               |
| Exact topic                | Distinctive phrase          |
| Message with an attachment | Attachment or Files filter  |
| Unread email               | Unread filter               |
| Flagged email              | Flagged filter              |
| Shared mailbox email       | Select shared account first |
| A specific file            | File name                   |
| Recent email               | Date filter where available |
  
**Search tips**  
* Start with sender or exact subject.  
* Add one filter at a time.  
* Check Focused and Other.  
* Check Archive, Deleted Items and Snoozed.  
* Confirm the selected account.  
* Compare Windows or web when mobile results appear incomplete.  
**Confirm the mailbox before searching and the result location before acting.**  
  
## Common touch gestures  

| Gesture            | Use                                     |
| ------------------ | --------------------------------------- |
| Tap                | Select a message or control             |
| Touch and hold     | Open contextual actions                 |
| Swipe              | Apply the configured message action     |
| Scroll             | Move through email, folders or settings |
| Drag               | Move or select where supported          |
| Pinch              | Zoom in supported previews              |
| Swipe up and pause | Open the app switcher                   |
  
**Swipe reminder**  
Your left and right swipe actions may differ from the training screenshots.  
Check:  
**Outlook Settings > Swipe Options**  
## Common swipe-action choices  
* Flag  
* Archive  
* Mark read or unread  
* Snooze  
* Move  
* Delete  
* None  
## Safety reminder  
A full swipe may complete an action immediately. Test swipe settings using low-risk messages.

| Gesture               | Use                                     |
| --------------------- | --------------------------------------- |
| One-finger movement   | Move the pointer                        |

**Important notes**  
* Outlook shortcuts depend on the active screen.  
* Standard editing shortcuts may work when Outlook-specific shortcuts do not.  
* Windows Control-key shortcuts may not match iPad Command-key shortcuts.  
* Test hardware in another approved app before treating the issue as an Outlook problem.
