## Video 5 — Search the correct account  
## Learning goal  
Search the primary or shared mailbox and verify the result location.  
## Scene sequence  
1. Confirm the primary account.  
2. Tap Search.  
3. Search “renewal.”  
4. Add an attachment filter.  
5. Open a result.  
6. Switch to the shared mailbox.  
7. Repeat the search.  
8. Confirm the result account.  
9. Clear the search.  
## End card  
**Confirm the account before searching and the result before acting.**
