## Video 7 — Summarize a thread or attachment  
## Learning goal  
Create and verify a Copilot summary.  
## Scene sequence  
1. Open a long thread.  
2. Select Summarize.  
3. Review the summary.  
4. Open a source reference.  
5. Confirm decision and deadline.  
6. Open a supported attachment.  
7. Select Summarize, where available.  
8. Compare the result with the original file.  
## End card  
**Use the summary to catch up. Use the source to confirm.**
