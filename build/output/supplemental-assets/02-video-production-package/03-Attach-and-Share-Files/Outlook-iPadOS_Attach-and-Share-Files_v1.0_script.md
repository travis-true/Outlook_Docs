## Video 3 — Attach and share a file safely  
## Learning goal  
Choose an approved file source and decide between a link and a copy.  
## Scene sequence  
1. Start a new message.  
2. Open the attachment menu.  
3. Choose OneDrive or SharePoint.  
4. Select a file.  
5. Share as a link.  
6. Review permissions.  
7. Show the Files app alternative.  
8. Show a managed share-sheet restriction.  
## End card  
**One current version → Share a link**
**Fixed version → Attach a copy**
