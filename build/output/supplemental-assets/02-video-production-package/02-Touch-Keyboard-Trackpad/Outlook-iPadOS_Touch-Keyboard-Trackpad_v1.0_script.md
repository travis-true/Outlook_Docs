## Video 2 — Use touch, Magic Keyboard and trackpad  
## Learning goal  
Use the primary input method for each type of Outlook task.  
## Scene sequence  
1. Tap a message.  
2. Touch and hold to open actions.  
3. Swipe to reveal a configured action.  
4. Use the trackpad pointer.  
5. Scroll with two fingers.  
6. Type using the Magic Keyboard.  
7. Press and hold Command to show available shortcuts.  
8. Switch applications.  
## End card  
**Touch for controls. Trackpad for precision. Keyboard for typing.**
