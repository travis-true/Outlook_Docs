## Page 7 — Take control workflow  
## Take control one decision at a time  
Most inbox challenges are not only about volume. They come from not knowing what still needs attention.  
Use this workflow whenever you open an email:  
## Email → Decide → Make visible → Follow through  
## Start with one question  
**Does this need action from me?**  
If the answer is yes, ask:  
1. **What is the next step?**  
2. **Who owns it?**  
3. **When does it need to happen?**  
4. **Where will I make it visible?**  
## Four doors for every email  

| Door | Use it when | What to do |
| ---------------------- | ------------------------------------- | ------------------------------------------------------- |
| Respond now | A quick response resolves the message | Reply and send |
| Do the task | The work can be completed quickly | Complete it, then file the email |
| Make follow-up visible | More work or waiting is required | Flag, Snooze, categorize, create a task or reserve time |
| Let it go | No action is required | File, archive or delete |
  
****Choose the right mobile follow-up tool****  

| Tool           | Best use                                     |
| -------------- | -------------------------------------------- |
| Flag           | Track unfinished work                        |
| Snooze         | Return a message later                       |
| Category       | Identify status, type or ownership           |
| Swipe action   | Complete a common action quickly             |
| To Do          | Track deadlines, reminders and several steps |
| My Day         | Focus on selected work today                 |
| Calendar event | Reserve focused work time                    |
| Meeting        | Coordinate or make a decision with others    |
  
****What to remember****  
* **Visible beats remembered.**  
* Use the fewest tools needed.  
* A swipe action changes the message quickly but does not replace judgment.  
* Snooze changes when the message appears but does not complete the work.  
* Copilot can help you move faster.  
* You remain responsible for the final message and action.  
**Copilot starts the draft. You finish the message.**

Some messages may remain available after the connection is lost.

* Whether Outlook synchronized the content  
* Whether the attachment was previously opened  
* App and account settings  
* Managed-device policy  
* Available device storage

You may be able to:

2. Write a reply or new message.

## Page 18 — Understand an open message  
## Understand an open message  
When you select or open an email, review these areas before responding.  

| Area           | Purpose                                        |
| -------------- | ---------------------------------------------- |
| Sender         | Identifies who sent the message                |
| To and Cc      | Shows who received it                          |
| Subject        | Identifies the conversation                    |
| Date and time  | Shows when the message was sent                |
| Conversation   | Displays earlier messages                      |
| Attachments    | Provides included files                        |
| Reply controls | Opens Reply, Reply All or Forward              |
| More options   | Displays Flag, Snooze, Move and other commands |
| Message body   | Contains the current content                   |
  
****Review the message****  
Before taking action:  
1. Confirm the sender.  
2. Review To and Cc.
