## Page 73 — Run a basic search  
## Search for email, people and files  
## Use this when  
Find Outlook content using a sender, recipient, subject, phrase or file name.  
## Open Search  
1. Open Outlook.  
2. Confirm the selected account.  
3. Tap **Search**.  
4. Tap the Search field.  
5. Enter a name, email address, subject, phrase or file name.  
6. Review the suggestions.  
7. Tap the correct result.  
## Useful search terms  

| Need                 | Search                             |
| -------------------- | ---------------------------------- |
| Renewal messages     | renewal                            |
| Messages from Jordan | Jordan White                       |
| Exact subject phrase | renewal effective date             |
| Proposal attachment  | Proposal Comparison                |
| Account topic        | [Approved account or project term] |
  
****Use specific terms****  
## Better  
Q3 renewal effective date  
## Less useful  
update  
## Clear the search  
1. Tap the clear control in the Search field.  
Or:  
1. Use the Back or close control.  
2. Confirm that the normal mailbox view returns.  
## Keyboard option  
When the Magic Keyboard is connected:  
1. Use the verified Search shortcut when available.  
2. Enter the search term.  
3. Use the arrow keys to review suggestions where supported.  
4. Press Return to select the result.

## Page 77 — Troubleshoot missing messages  
## Check the selected account  
1. Open the account menu.  
2. Confirm the primary or shared mailbox.  
3. Open the expected folder.  
4. Search again.  
## Check common locations  
Review:  
* Focused  
* Other  
* Archive  
* Deleted Items  
* Snoozed  
* Drafts  
* Sent  
* Existing folders  
* Shared mailboxes  
## Check Search  
1. Clear the current search.  
2. Verify spelling.  
3. Search by sender.  
4. Search by exact subject.  
5. Search by file name.  
6. Remove unnecessary filters.  
7. Expand the date range where supported.  
## Check connectivity  
1. Confirm Wi-Fi or cellular connectivity.  
2. Refresh Outlook.  
3. Close and reopen the app.  
4. Wait for synchronization.  
5. Search again.  
## Check another platform  
Use Outlook on the web or Windows when:  
* Mobile Search appears incomplete  
* Older messages do not appear  
* The Online Archive is required  
* Complex folder or rule behavior may have moved the message  
* Shared-mailbox Search remains incomplete  
## Get help when  
* A recently received message is missing across platforms  
* Several folders fail to update  
* Search repeatedly returns incomplete results  
* Shared mailbox content does not synchronize  
* The issue may involve retention or deletion requirements  
## Support information to provide  
* Account or mailbox  
* Sender  
* Subject  
* Approximate date  
* Last known folder  
* Search terms  
* Connectivity status  
* Screenshot with sensitive information removed  
* Whether the message appears on Windows or the web  
## What to remember  
A missing message may be sorted, moved, Snoozed, archived or deleted.

## Page 80 — Review time zone and duration  
## Choose a realistic duration  

| Work type           | Example duration       |
| ------------------- | ---------------------- |
| Quick review        | 15 minutes             |
| Draft a response    | 20–30 minutes          |
| Account research    | 30–60 minutes          |
| Proposal comparison | 45–60 minutes          |
| Complex work        | Several focused blocks |
  
****Review the time zone****  
## Use this when  
Travel, field work or remote participants may affect the displayed time.  
1. Review the event date and time.  
2. Check the time zone shown by Outlook.  
3. Confirm the iPad date and time.  
4. Review the recipient or business location when relevant.  
5. Correct the event before saving.  
## Review event details  
Before saving:  
* Title describes the action  
* Date is correct  
* Start and end times are realistic  
* Reminder is useful  
* Notes include only needed context  
* No attendees are included unless required  
## Add another time block when needed  
For complex work:  
1. Create the first focused event.  
2. Define the expected outcome.  
3. Create another event only when the work truly requires it.  
4. Avoid reserving one unrealistically long block.  
## What to remember  
A realistic time block is more useful than a vague all-day reminder.  
## Avoid this  
* Do not reserve more time than the task requires.  
* Do not ignore time-zone changes while traveling.  
* Do not use an all-day event for short focused work.  
* Do not assume the event completes the source email.

## Use this when  
The next action requires discussion, coordination or a decision involving other people.  
## Decide whether a meeting is needed  
Use a meeting when:  
* Several people must make a decision  
* Discussion is more efficient than repeated email  
* Participants need to coordinate  
* A shared outcome is required  
Use email when:  
* One clear answer will resolve the issue  
* No discussion is required  
* A written update is enough

5. Add optional attendees when appropriate.  
6. Choose the date and time.

8. Add a short agenda.  
9. Review the invitation.

When the event or meeting time arrives:  
1. Open the calendar item.  
2. Review the expected outcome.  
3. Open the source email when needed.  
4. Complete the work or discussion.  
5. Record the confirmed result.  
## Return to the source email  
After completing the action:  
1. Open Outlook Mail.  
2. Locate the source email.  
3. Send the required response.  
4. Update the task.  
5. Remove or complete the flag.  
6. Update the category.  
7. Archive or file the message.  
## Example  
**Event:** Review proposal comparison and reply to producer  
**During the event:**  
* Review the attachment  
* Confirm the effective date  
* Identify open questions  
* Draft the response  
**After the event:**  
* Send the response  
* Complete the To Do task  
* Remove the flag  
* Archive the message  
## What to remember  
The calendar reserves time. Returning to the email and completing the business action closes the loop.  
## Avoid this  
* Do not assume the event communicates progress to the sender.  
* Do not leave the source email flagged after completion.  
* Do not leave closed work in My Day.  
* Do not archive before sending the required response.

## Update the mobile signature  
## Use this when  
Add approved identifying information to messages sent from Outlook on iPad.  
1. Open **Outlook Settings**.  
2. Tap **Signature**.  
3. Confirm whether the signature applies to all accounts or one account.  
4. Enter the approved signature.  
5. Review spelling and contact information.  
6. Return to Settings.  
7. Create a test message.  
## Shared mailbox caution  
When sending from a shared mailbox:  
* Confirm which signature appears.  
* Remove an individual signature when a team signature is required.  
* Insert the approved shared signature manually when needed.  
## Set automatic replies  
When supported:  
1. Open **Outlook Settings**.  
2. Select the company work account.  
3. Tap **Automatic Replies**.  
4. Turn the setting on.  
5. Enter the approved message.  
6. Add an end time when available.  
7. Review external-reply settings where supported.  
8. Save the setting.  
## Suggested automatic reply  
Thank you for your message. I am unavailable and will return on [date]. For urgent assistance, contact [approved team or contact]. Otherwise, I will respond after I return.  
## Complete on Windows or web  
Use another Outlook platform when you need:  
* Different internal and external messages  
* Advanced scheduling  
* Meeting-decline options  
* Detailed shared-mailbox automatic replies  
* More signature formatting  
## What to remember  
Review the signature and sending account before every shared-mailbox message.  
## Avoid this  
* Do not add unapproved logos or quotations.  
* Do not include unnecessary travel details.  
* Do not name an alternate contact without confirming availability.  
* Do not leave automatic replies active after returning.

## Page 91 — Configure text size and accessibility  
## Improve readability  
## Use this when  
Outlook text, controls or message content are difficult to read or use.  
## Adjust text size in iPadOS  
1. Open **Settings**.  
2. Tap **Display & Brightness**.  
3. Tap **Text Size**.  
4. Adjust the slider.  
5. Return to Outlook.  
6. Review the result.  
## Use Accessibility settings  
1. Open **Settings**.  
2. Tap **Accessibility**.  
3. Review approved options such as:  
    * Larger Text  
    * Bold Text  
    * Display and Text Size  
    * VoiceOver  
    * Zoom  
    * Pointer Control  
    * Keyboard settings  
4. Adjust only the setting you need.  
5. Return to Outlook.  
## Use VoiceOver awareness  
Outlook and Copilot controls may be used with VoiceOver.  
When supporting VoiceOver users:  
* Use descriptive labels  
* Keep procedures in live text  
* Avoid color-only directions  
* Describe the control purpose  
* Verify focus order  
## Use landscape or portrait strategically  
* Landscape may provide more workspace.  
* Portrait may make the message body easier to focus on.  
* Choose the orientation that preserves readability.  
## What to remember  
Increase readability rather than shrinking text to fit more content.  
## Avoid this  
* Do not reduce text until it becomes difficult to read.  
* Do not depend on screenshots as the only instruction.  
* Do not change several accessibility settings at once while troubleshooting.  
* Do not assume everyone uses the same display configuration.

## Page 92 — Configure keyboard and trackpad settings  
## Open keyboard settings  
1. Open iPad **Settings**.  
2. Tap **General**.  
3. Tap **Keyboard**.  
4. Review available settings.  
5. Change only the setting needed.  
6. Return to Outlook.  
Settings may include:  
* Auto-Correction  
* Predictive text  
* Smart punctuation  
* Hardware Keyboard  
* Keyboard shortcuts  
* Key repeat or accessibility controls  
## Open trackpad settings  
1. Open **Settings**.  
2. Tap **General**.  
3. Tap **Trackpad** or **Trackpad & Mouse**.  
4. Review:  
    * Tracking speed  
    * Natural scrolling  
    * Tap to Click  
    * Secondary Click  
5. Make the approved change.  
6. Return to Outlook.  
7. Test the pointer and gesture.  
## If the Magic Keyboard does not respond  
1. Confirm that the iPad is seated correctly.  
2. Remove and reconnect it.  
3. Test typing in another approved app.  
4. Restart the iPad.  
5. Review keyboard and trackpad settings.  
6. Contact device support if the issue continues.  
## Organization managed  
Some keyboard, accessibility or input settings may be controlled or limited by device policy.  
## What to remember  
Test the keyboard or trackpad in another approved app before assuming Outlook is the problem.  
## Avoid this  
* Do not change unfamiliar hardware settings during an Outlook issue.  
* Do not force or damage the connector.  
* Do not assume Windows right-click behavior applies exactly.  
* Do not install unapproved keyboard software.
