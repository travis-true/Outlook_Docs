## Guide purpose  
This guide will help sales employees manage email and email-related follow-up from a company-issued iPad.  
It will explain how to:  
* Navigate Outlook using touch, keyboard and trackpad controls  
* Read, create and respond to email  
* Attach and share files  
* Make follow-up visible  
* Search and organize messages  
* Use email-related calendar follow-up  
* Work with shared mailboxes  
* Use supported Copilot features  
* Understand which tasks require Windows or Outlook on the web  
* Resolve common Outlook, iPadOS and managed-device issues  
  
## Core workflow  
The guide will preserve the established session workflow:  
## Email → Decide → Make visible → Follow through  
Employees will use these questions:  
1. **Does this need action from me?**  
2. **What is the next step?**  
3. **Who owns it?**  
4. **When does it need to happen?**  
5. **Where will I make it visible?**  
The four-door model remains:  
* Respond now  
* Do the task  
* Make follow-up visible  
* Let it go  
The participant workbook establishes the same action-first model, “Visible beats remembered” principle and expectation that employees review Copilot output before using it.

## Availability may vary
