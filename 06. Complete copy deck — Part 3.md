## Page 49 — Decide what the email needs  
## Decide what the email needs  
## Use this when  
You open an email and need to determine what should happen next.  
## Start with one question  
**Does this need action from me?**  
## If no action is required  
Choose the appropriate action:  
* Leave it temporarily if you still need to read it  
* Move it to a reference folder  
* Archive it  
* Delete it when it has no continuing value  
## If action is required  
Ask:  
1. What is the next step?  
2. Who owns it?  
3. When does it need to happen?  
4. Where will I make it visible?  
## Choose one of four doors  

| Door | Use when | Action |
| ---------------------- | ------------------------------------ | ------------------------------------------------------- |
| Respond now | A quick response completes your part | Reply and send |
| Do the task | The work can be completed quickly | Complete it, then file the email |
| Make follow-up visible | More work or waiting is required | Flag, Snooze, categorize, create a task or reserve time |
| Let it go | No action is required | File, archive or delete |
  
****Example — Waiting for information****  
**Message:** A producer asks for an updated proposal, but the effective date is missing.  
**Next step:** Ask the producer to confirm the date.  
**Follow-up plan:**  
* Send the clarification request  
* Flag the email  
* Apply the Waiting category where supported  
* Add a reminder in To Do  
## Example — Work that needs focus  
**Message:** A proposal must be reviewed before Friday.  
**Next step:** Compare the proposal with the current account information.  
**Follow-up plan:**  
* Flag the email  
* Create a To Do task  
* Add review steps  
* Reserve 30 minutes on the calendar  
## What to remember  
“Handle this later” is not a next action. Name what you will do.  
## Avoid this  
* Do not move unfinished work without tracking it.  
* Do not Snooze a message only to avoid making a decision.  
* Do not create several duplicate reminders.  
* Do not rely on memory for field follow-up.

## Page 50 — Choose the right mobile follow-up tool  
## Choose the right mobile follow-up tool  

| Need | Best tool | What it does |
| ---------------------------------- | ----------------- | --------------------------------------------------- |
| Track unfinished email | Flag | Marks the message for follow-up |
| Return the email later | Snooze | Removes it temporarily and brings it back |
| Identify status or work type | Category | Adds a named label where supported |
| Complete a frequent action quickly | Swipe | Applies the configured action |
| Track several steps | Microsoft To Do | Adds dates, reminders, notes and steps |
| Focus on selected work today | My Day | Places chosen tasks in today’s list |
| Reserve focused time | Calendar event | Protects time for the work |
| Coordinate with others | Meeting | Supports discussion or a shared decision |
| Store completed information | Folder or Archive | Removes completed reference material from the Inbox |
  
****Use Flag when****  
* The email remains unfinished  
* You need to revisit the source message  
* You want the email connected to To Do  
## Use Snooze when  
* The message is not useful until a future time  
* You want it temporarily removed from the Inbox  
* You know when it should return  
## Use Category when  
* You need to identify status, work type or ownership  
* The team uses agreed labels  
* The category is visible and understood  
## Use Swipe when  
* The action is frequent  
* You understand the configured result  
* The action does not require additional judgment  
## Use To Do when  
* The work includes several steps  
* You need a due date or reminder  
* You want to add the task to My Day  
## Use Calendar when  
* The work needs protected time  
* It must happen on a specific date  
* A meeting is required  
## What to remember  
Choose the tool based on what must happen next, not which control is easiest to reach.  
## Avoid this  
* Do not combine every tool on one email.  
* Do not use read or unread status as a complete follow-up system.  
* Do not configure a destructive swipe action without understanding it.  
* Do not create duplicate tasks for the same work.

## Page 51 — Flag a message  
## Flag a message  
## Use this when  
Mark an email as unfinished work that you need to revisit.  
## Flag an open message  
1. Open the message.  
2. Tap **More options**.  
3. Tap **Flag**.  
4. Confirm that the flag indicator appears.  
## Flag from the message list  
Depending on the current layout:  
1. Touch and hold the message.  
Or:  
1. Open the message menu.  
2. Tap **Flag**.  
## Flag with a swipe action  
When Flag is assigned to a swipe:  
1. Swipe the message in the configured direction.  
2. Stop when the Flag action appears.  
3. Complete the swipe.  
4. Confirm that the flag appears.  
## Find flagged messages  
Use one or more of these methods:  
* Search and apply the Flagged filter  
* Open Microsoft To Do  
* Open the Flagged Email list  
* Review the flag indicator in the message list

## Page 52 — Unflag or complete follow-up  
## Remove a flag  
## Use this when  
The flag was added by mistake or is no longer needed.  
1. Open or select the flagged message.  
2. Open **More options**.  
3. Tap **Unflag**, **Clear flag** or the available flag control.  
4. Confirm that the flag is removed.  
## Complete the follow-up  
When the work is finished:  
1. Complete the required action.  
2. Send the response when needed.  
3. Open the related task in To Do.  
4. Mark the task complete.  
5. Return to Outlook.  
6. Remove or complete the flag where supported.  
7. Update the category.  
8. Archive or file the message.  
## Complete or clear?  

| Action          | Use when                              |
| --------------- | ------------------------------------- |
| Complete        | The required work is finished         |
| Clear or unflag | The flag was unnecessary or incorrect |

Flags in a shared mailbox may not appear in your primary Flagged Email list in Microsoft To Do.  
Follow the team’s approved shared-mailbox process instead.

Do not remove the follow-up marker until the business action is complete.  
## Avoid this  
* Do not unflag work only to clean up the Inbox.  
* Do not mark a task complete before sending the required response.  
* Do not assume a shared-mailbox flag creates personal ownership.  
* Do not leave the email and task showing conflicting statuses.

## Page 53 — Snooze a message  
## Snooze a message  
## Use this when  
Hide an email temporarily and return it to the Inbox at a more useful date and time.  
## Snooze an open message  
1. Open the message.  
2. Tap **More options**.  
3. Tap **Snooze** or **Schedule**.  
4. Choose a suggested time.  
Or:  
1. Choose a custom date.  
2. Choose a return time.  
3. Confirm the selection.  
## Snooze from the message list  
Depending on the layout:  
1. Touch and hold the message.  
2. Tap **Snooze**.  
Or use a configured swipe action when available.  
## Choose a useful return time  
Choose a time when:  
* You expect the needed information  
* The work becomes relevant  
* You will be available to act  
* The deadline still allows enough time

## What happens next  
* The message leaves the current Inbox view.  
* Outlook stores it in the Snoozed location.  
* It returns at the selected time.  
* It may appear as a new message when it returns.

Snooze changes when you see the email. It does not complete the work.

* Do not repeatedly Snooze a message without changing the plan.  
* Do not use Snooze when a task or calendar block is more appropriate.

## Page 54 — Find, change or cancel Snooze  
## Find a Snoozed message  
1. Open the account and folder menu.  
2. Tap **Snoozed** or the available scheduled-message folder.  
3. Locate the message.  
4. Open it to review the selected return time.  
## Change the return time  
1. Open or select the Snoozed message.  
2. Open **More options**.  
3. Tap **Snooze**.  
4. Choose a new date and time.  
5. Confirm the change.  
## Cancel Snooze  
1. Open the Snoozed location.  
2. Select the message.  
3. Open the Snooze menu.  
4. Choose the option to return it to the Inbox or cancel the scheduled return.  
5. Confirm that the message returns.  
The exact folder and command names must be validated in the managed Outlook version.  
## Check the time zone  
If the message returns at an unexpected time:  
1. Review the selected return time.  
2. Confirm the iPad’s date and time.  
3. Confirm the Outlook time zone where available.  
4. Reschedule the message.  
## Compare Flag and Snooze  

| Tool | Message remains visible? | Returns later? | Tracks unfinished work? |
| ------ | ------------------------ | -------------- | ----------------------- |
| Flag | Usually | No | Yes |
| Snooze | No | Yes | Not by itself |
  
****What to remember****  
When a Snoozed message returns, make a new decision instead of automatically Snoozing it again.  
## Avoid this  
* Do not assume a missing message was deleted before checking Snoozed.  
* Do not Snooze shared work without team coordination.  
* Do not ignore time-zone differences.  
* Do not use Snooze to hide an overloaded Inbox.

## Page 55 — Use Pin where available  
## Pin a message  
## Use this when  
Keep a message temporarily visible near the top of the message list.  
## When Pin is available  
1. Select or open the message.  
2. Open **More options**.  
3. Tap **Pin**.  
4. Confirm that the message moves to the pinned area.  
To remove it:  
1. Open the message menu.  
2. Tap **Unpin**.  
## What Pin does  
* Keeps a message visually prominent  
* Changes its position  
* Helps with short-term visibility  
## What Pin does not do  
* Create a deadline  
* Create a reminder  
* Assign ownership  
* Complete the work  
* Reserve calendar time  
## If Pin is unavailable  
Use:  
* Flag for unfinished work  
* Snooze to bring the message back later  
* To Do for a due date or reminder  
Or complete the Pin task in New Outlook or Outlook on the web.  
## Availability may vary  
Pin must be validated in the current managed Outlook app before publication.  
## What to remember  
Pin is a visibility tool, not a complete follow-up plan.  
## Avoid this  
* Do not leave many messages pinned.  
* Do not use Pin as the only signal for time-sensitive work.  
* Do not assume a pinned message is also flagged.  
* Do not leave completed messages pinned.

## Page 59 — Use swipe actions safely  
## Use a swipe action  
1. Open the message list.  
2. Locate the message.  
3. Swipe in the configured direction.  
4. Pause when the action appears.  
5. Confirm the action.  
6. Complete the swipe when ready.  
7. Review the result.  
## Short swipe versus full swipe  
Depending on the current app behavior:  
* A short swipe may reveal the action  
* A full swipe may complete the action immediately  
Test with a low-risk message before relying on the gesture.  
## Confirm the result  
After swiping, check:  
* Message location  
* Flag status  
* Read or unread status  
* Snooze return  
* Archive or Deleted Items  
## Correct an accidental action  
When an Undo notification appears:  
1. Tap **Undo** immediately.  
If Undo is unavailable:  
* Check Archive  
* Check Deleted Items  
* Check Snoozed  
* Search for the message  
* Restore or move it as needed  
## What to remember  
Know the configured action before swiping.  
## Avoid this  
* Do not swipe quickly through unfamiliar messages.  
* Do not use full-swipe behavior before testing it.  
* Do not assume a message disappeared permanently.  
* Do not use unread status as ownership in shared work.

* Review Flagged Email  
* Review My Day  
* Review Snoozed messages that returned  
* Reschedule unfinished work  
* Complete closed tasks  
* Confirm important messages were sent

A visible reminder helps you return to the work. Closing the loop completes it.  
## Avoid this

* Do not leave closed tasks in My Day.  
* Do not archive before sending the required response.

## Page 71 — Complete organization tasks on Windows or the web  
## Use another Outlook platform for advanced organization  
Outlook on iPadOS is best for reading, responding, moving, archiving and timely mobile follow-up.  
Use Windows or Outlook on the web when the task needs more control.  

| Task | Recommended platform |
| ------------------------------------- | -------------------------------------------- |
| Create or edit rules | New Outlook or Outlook on the web |
| Reorder or troubleshoot rules | Windows or web |
| Use Sweep | Outlook on the web or New Outlook |
| Create or manage Quick Steps | Windows or web where supported |
| Manage Search Folders | Classic Outlook or supported Windows version |
| Create or restructure several folders | Windows or web |
| Perform large-scale mailbox cleanup | Windows or web |
| Manage advanced categories | Windows or web |
| Configure shared-mailbox automation | Windows or web |
  
****Before switching platforms****  
1. Confirm that mobile changes synchronized.  
2. Save unfinished drafts.  
3. Note the task you need to complete.  
4. Open the appropriate Outlook platform.  
5. Confirm the same work account and mailbox.  
6. Complete the advanced task.  
7. Return to the iPad and confirm synchronization.  
## Core reminder  
Use the iPad for timely field work. Use Windows or the web for detailed mailbox administration.  
## Avoid this  
* Do not attempt to work around unavailable mobile controls.  
* Do not assume every Windows feature has a mobile equivalent.  
* Do not make mailbox-wide changes without understanding their impact.  
* Do not create duplicate organization systems across platforms.
