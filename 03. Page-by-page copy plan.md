## Page 7 — Take control workflow  
**Purpose:** Establish the decision model.  
**Core copy:**  
**Email → Decide → Make visible → Follow through**  
**Does this need action from me?**  
If yes:  
1. What is the next step?  
2. Who owns it?  
3. When does it need to happen?  
4. Where will I make it visible?  
**Four-door table:**  

| Door | Action |
| ---------------------- | ------------------------------------------------------- |
| Respond now | Send a quick reply |
| Do the task | Complete it if it is fast |
| Make follow-up visible | Flag, Snooze, categorize, create a task or reserve time |
| Let it go | File, archive or delete |
  
**Follow-up comparison:**  
* Flag — Track unfinished work  
* Snooze — Return later  
* Category — Identify status or type  
* To Do — Track steps, dates and reminders  
* My Day — Focus on selected work today  
* Calendar — Reserve time  
**Reminders:**  
* Visible beats remembered.  
* Use the fewest tools needed.  
* Copilot starts the draft. You finish the message.
