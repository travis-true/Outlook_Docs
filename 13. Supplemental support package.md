## Asset 3 — iPad quick-fixes card  
## Asset overview  
**Title:** Outlook on iPadOS: Quick fixes **Format:** One-page job aid or two-sided desk card **Estimated use time:** 1–3 minutes  
## Copy deck  
## Outlook on iPadOS: Quick fixes  
## Start with five checks  
Confirm:  
1. Company-issued iPad  
2. Wi-Fi or cellular connection  
3. Microsoft Outlook app  
4. Company work account  
5. Correct primary or shared mailbox  
  
## Outlook will not open or refresh  
Try:  
1. Confirm the connection.  
2. Open Outlook once.  
3. Refresh the message list.  
4. Return to the Home Screen and reopen Outlook.  
5. Close Outlook from the app switcher only if it is frozen.  
6. Restart the iPad when needed.  
Do not reinstall Outlook or remove the managed account unless support directs you.  
  
## Outlook keeps asking you to sign in  
Check:  
* Correct company account  
* Device date and time  
* Approved authentication app  
* Network connection  
* Whether other Microsoft 365 apps also request sign-in  
Then:  
1. Complete the approved sign-in prompt.  
2. Lock and unlock the iPad.  
3. Reopen Outlook.  
4. Restart the iPad.  
5. Contact identity support when the prompts continue.  
  
## Notifications do not appear  
Check both layers:  
## Outlook  
**Outlook Settings > Notifications**  
## iPadOS  
**Settings > Notifications > Outlook**  
Also review:  
* Focus modes  
* Lock Screen alerts  
* Banners  
* Sounds  
* Badges  
  
## The Magic Keyboard or trackpad does not work  
1. Confirm that the iPad is seated correctly.  
2. Remove and reconnect the keyboard.  
3. Test typing in another approved application.  
4. Review **Settings > General > Keyboard**.  
5. Review **Settings > General > Trackpad**.  
6. Restart the iPad.  
Contact device support if the issue affects multiple apps.  
  
## A swipe did the wrong thing  
1. Tap **Undo** when available.  
2. Check Archive, Deleted Items or Snoozed.  
3. Open **Outlook Settings > Swipe Options**.  
4. Confirm the left and right actions.  
5. Test them with low-risk messages.  
  
## A file will not open or share  
Check:  
* Connection  
* File type  
* Approved app  
* OneDrive or SharePoint access  
* Jamf restriction  
* File permissions  
Do not copy the file into a personal app or storage location.  
  
## Get help when  
* Outlook remains unavailable after restart  
* Authentication repeatedly fails  
* Managed sharing blocks required work  
* The work account is missing  
* The shared mailbox does not appear  
* The issue affects several managed apps  
## Support details to include  
* iPad model  
* iPadOS version  
* Outlook version  
* Account and mailbox  
* Network type  
* Exact error  
* Date and time  
* Screenshot with sensitive information removed  
* Steps already attempted

## Shared mailboxes in Outlook on iPadOS  
## Before you start  
You need approved access.  
Permissions may include:  
* Read and manage  
* Send As  
* Send on Behalf  
Adding the mailbox displays it. It does not grant permission.  
  
## Add the shared mailbox  
1. Open Outlook.  
2. Tap the account or profile control.  
3. Tap **Add Account**.  
4. Tap **Add a Shared Mailbox**.  
5. Enter the approved mailbox name or address.  
6. Select the correct result.  
7. Wait for Outlook to add it.  
8. Tap the shared mailbox.  
9. Open **Inbox**.  
  
## Switch between mailboxes  
1. Open the account menu.  
2. Tap the primary or shared mailbox.  
3. Confirm the active account.  
4. Open the required folder.  
Before acting, confirm:  
* Mailbox  
* Folder  
* Message  
* Sending account  
  
## Review shared folders  
Check:  
* Shared Inbox  
* Shared Drafts  
* Shared Sent  
* Team folders  
* Deleted Items  
Before replying, review Shared Sent to see whether someone already responded.  
  
## Send from the shared account  
1. Open the shared mailbox.  
2. Tap **Compose**.  
3. Confirm the shared sending account.  
4. Add recipients.  
5. Enter a clear subject.  
6. Write the message.  
7. Add the approved shared signature.  
8. Review attachments and links.  
9. Confirm the shared account again.  
10. Tap **Send**.

## Copilot is missing  
Check:  
1. Correct work account  
2. Supported primary mailbox  
3. Internet connection  
4. Current Outlook version  
5. Another supported conversation  
6. Copilot available on Windows or the web  
7. Feature enabled for the account  
  
## A thread will not summarize  
Check:  
* Message is in the primary mailbox  
* Conversation contains enough content  
* Outlook is connected  
* Message protection is supported  
* Another thread can be summarized  
Try:  
1. Reopen the conversation.  
2. Refresh Outlook.  
3. Close and reopen the app.  
4. Test another low-risk conversation.  
5. Compare the feature on Windows or the web.  
  
## An attachment will not summarize  
Check:  
* File is PDF, Word or PowerPoint  
* File opens normally  
* Email is in the supported mailbox  
* File protection is supported  
* Mobile attachment summaries are enabled  
Use Windows or Outlook on the web when the mobile control is unavailable.  
  
## Draft with Copilot does not work  
Improve the prompt.  
Include:  
* Goal  
* Context  
* Source  
* Expectation  
Example:  
Draft a warm professional reply asking for the missing effective date. Keep it under 100 words and do not add new facts.  
Avoid:  
Reply to this.  
  
## Coaching is missing  
Check:  
* Draft contains enough text  
* Copilot control is available  
* Work account is selected  
* Outlook is connected  
* Feature is enabled  
  
## Copilot does not use shared-mailbox content  
Copilot inside Outlook may be limited to the supported primary mailbox.  
Do not assume it is using:  
* Shared mailbox email  
* Delegate mailbox email  
* Archive content  
* Every attachment  
Use the approved Windows, web or Microsoft 365 Copilot workflow when shared-mailbox support is required.  
  
## Copilot output contains incorrect information  
1. Stop before sending or acting.  
2. Open the source.  
3. Review citations or original files.  
4. Remove unsupported statements.  
5. Correct names, dates, owners and deadlines.  
6. Improve the prompt when regenerating.  
7. Complete the final review yourself.
